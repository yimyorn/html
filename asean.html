ប្រវត្តិអាស៊ាន


អាស៊ាន គឺជាសមាគមប្រជាជាតិអាស៊ីអាគ្នេយ៍។ អាស៊ានបង្កើតដំបូងមាន ៥ ប្រទេសចូលរួម មានដូចជា ៖

​​​​

-ប្រទេសថៃ (០៨ សីហា ១៩៩៧)

-ប្រទេសសិង្ហបុរី ( ០​៨ សីហា ១៩៦៧ )

-ប្រទេសម៉ាឡេស៊ី ( ០៨ សីហា​ ១៩៦៧​​​ )

-ប្រទេសហ្វីលីពីន ( ០៨ សីហា ១៩៦៧ )

-ប្រទេសឥណ្ឌូនេស៊ី ( ០៨ សីហា ១៩៦៧​)

អាស៊ានបង្កើតដំបូងនៅ ថ្ងៃទី ០៨ សីហា ឆ្នាំ ១៩៦៧ ។ នៅទីក្រុងបាងកក ប្រទេសថៃ។ ​
+បើ​យោងតាម​ធម្មនុញ្ញ​អាស៊ាន ទង់​អាស៊ាន តំណាង​ឲ្យ​អាស៊ាន​មួយ​មាន​ស្ថិរភាព សន្តិភាព ការរួបរួម និង​ថាមពល​។
ពណ៌​របស់​ទង់​អាស៊ាន​មានៈ ពណ៌​ខៀវ ក្រហម ស និង​ពណ៌​លឿង ដែល​តំណាង​ឲ្យ ពណ៌​សំខាន់ៗ​នៃ​គ្រប់​ទង់ជាតិ​របស់​ប្រទេស​សមាជិក​អាស៊ាន​ទាំងអស់​។​


+ពណ៍នៃទង់ជាតិអាស៊ានទាំងអស់តំណាឲ្យ ៖:😃 ​
ពណ៌​ខៀវ ៖ តំណាង​ឲ្យ​សន្តិភាព និង​ស្ថិរភាព​ ​
ពណ៌​ក្រហម ៖ បង្កប់​អត្ថន័យ​នៃ​ភាពក្លាហាន និង​ថាមពល ​
ពណ៌​ស ៖​បង្ហាញ​នូវ​ភាព​បរិសុទ្ធ​ស្អាតស្អំ និង​ ​
ពណ៌​លឿង ៖ ជា​និមិត្តរូប​នៃ​វិបុលភាព​
រូប​កួរស្រូវ ​៖ តំណាង​ឲ្យ​សុ​បិ​ន្ត​របស់​បិតា​ស្ថាបនិក​អាស៊ាន ដែល​ចង់បាន​អាស៊ាន​មួយ​មាន​គ្រប់​ប្រទេស ទាំងអស់ នៅក្នុង​តំបន់​អាស៊ីអាគ្នេយ៍​រួបរួមគ្នា ក្នង​មិត្តភាព​និង​សាមគ្គីភាព​៕


ចាប់​ពី​ឆ្នាំ​១៩៨៤ មក ទើប​បណ្ដា​ប្រទេស​ដទៃ​ប្រាំ​ទៀត​ក្នុងតំបន់​នេះ រួម​មាន ប៊្រុយណេ( ៧ មករា ១៩៨៤ ) វៀតណាម (២៨ កក្កដា ១៩៩៥) ឡាវ (២៣ កក្កដា ១៩៩៧)​ មីយ៉ាន់ម៉ា( ២៣ កក្កដា ១៩៩៧ ) និង​កម្ពុជា(​៣០ មេសា​១៩៩៩) ដែល​ជា​កូនពៅ បាន​ក្លាយ​ជា​សមាជិក​នៃ​សមាគម​នេះ​ជា​បន្តបន្ទាប់។
+គោលបំណងនៃការបង្កើតអាស៊ានគឺ ៖:😄
-ធ្វើឱ្យតំបន់សេដ្ឋកិច្ចមានការប្រកួតប្រជែងខ្ពស់
-ធ្វើឱ្យតំបន់នៃការអភិវឌ្ឍសេដ្ឋកិច្ចប្រកបដោយសមធម៌
-បង្កើតកិច្ចសហប្រតិបត្តិការសេដ្ឋកិច្ចនយោបាយ សង្គម និងវប្បធម៌
-មានកំណើនសេដ្ឋកិច្ច និងការអភិវឌ្ឍសង្គម និងវប្បធម៌ក្នុងនិងបណ្តាប្រទេសនៃ តំបន់អាស៊ីអាគ្នេយ៍
-ការពារផលប្រយោជន៍គ្នាទៅវិញទៅមក សន្តិភាព និងស្ថិរភាពក្នុងតំបន់...។
+គុណសម្បត្តិនៃការចូលជាសមាជិកអាស៊ាន:😎
- មានការរីកចម្រើនផ្នែកទីផ្សារ
- លែងមានការព្រួយបារម្មណ៍ចំពោះបញ្ហា សន្តិសុខ
-​​ មានសន្តិភាព សេរីភាពក្នុងតំបន់ ដោះស្រាយជម្លោះរួម
- ផ្លាស់ប្ដូរធនធានមនុស្ស វិទ្យាសាស្ត្រ បច្ចេកវិទ្យាថ្មីៗ
- ទទូលបានការកម្ចីប្រាក់ពី បណ្ដាប្រទេសទាំង ៩ និង ធនាគារមានការប្រាក់ទាប
- កម្ពុជាលេចមុខក្នុងឆាកអន្តរជាតិ
